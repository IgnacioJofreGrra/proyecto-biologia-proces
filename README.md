# proyecto-biologia-proces

## Archivo único listo para compartir

Se agregó `index-single.html`, una versión autoincluida que puedes abrir desde cualquier carpeta o equipo sin necesidad de la estructura original.

- CSS y JS están embebidos dentro del archivo.
- Las imágenes se cargan desde GitHub (URLs raw). Requiere conexión a Internet para mostrarlas. Si lo necesitas 100% offline, pide la versión con las imágenes embebidas en Base64 y la generamos.

### Cómo usar

1. Abre `index-single.html` con tu navegador.
2. Usa los botones ❮ y ❯ para navegar las diapositivas. En las preguntas, el botón “Siguiente” aparece tras responder.

### Estructura original

El proyecto original permanece igual (`index.html`, `assets/`, `images/`). El nuevo archivo único no altera la funcionalidad existente.
